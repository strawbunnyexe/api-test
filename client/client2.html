Homepage

Get Pokemon (GET, HEAD)

Query Params

  • Name - A pokemon's name to search by

Returns

  • A JSON pokemon object {name: string, type: [string], rating: number}

Get Pokemon Type (GET, HEAD)

Query Params

  • Type - A pokemon type to filter by

Returns

  • A JSON array of pokemon that match type {name: string}

Get Pokemon Evolutions (GET, HEAD)

Query Params

  • Name - A pokemon's name to search by

Returns

  • A JSON pokemon object of the given pokemon's next evolution if it exists {name: string}

Get All Pokemon (GET, HEAD)

Query Params

  • No Supported Params

Returns

  • A JSON array of pokemon objects

Add Pokemon (POST)

Supported Body Formats

  • application/json, application/x-www-form-urlencoded

Body Params

  • Name (string) - A pokemon's name to add
  • Type (string) - A pokemon type to add with new pokemon

Returns

  • A JSON pokemon object representing new pokemon {name: string, type: [string]}

Rate Pokemon (POST)

Supported Body Formats

  • application/json, application/x-www-form-urlencoded

Body Params

  • Name (string)- A pokemon's name to search by
  • Rating (number) - A rating to give to pokemon (1-5)

Returns

  • A JSON pokemon object representing updated pokemon {name: string, type: [string], rating: number}
